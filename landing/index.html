Everything You Need for Email

A complete self-hosted email platform — from receiving mail to sending via SMTP relay, hosting landing pages, and managing everything through a clean admin panel.

📧 Inbound Email

Receive emails on your custom domains with catch-all mailboxes. Full MIME parsing with attachment support, HTML sanitization, email threading with reply tracking, and real-time inbox delivery.

📤 SMTP Relay & Submission

Send emails via SMTP AUTH on ports 587 (STARTTLS) and 465 (implicit TLS). Use API keys as SMTP credentials, automatic DKIM signing for every outbound message, direct MX delivery or upstream relay, with configurable daily send limits.

🌐 Static Website Hosting

Upload ZIP archives and publish instantly to a subdomain. Bind custom domains with automatic Let's Encrypt SSL via Traefik. Atomic staging-to-live publishing with rollback, ZIP bomb protection, and thumbnail generation.

🔑 API Access

Manage API keys with scoped permissions (send_email, full_access). Keys are SHA-256 hashed at rest and revealed only once upon creation. Use them for SMTP auth or REST API calls with rate limiting and usage counters.

⚡ Real-time Updates

Server-Sent Events (SSE) push new email notifications to your browser instantly via Redis pub/sub. No polling needed — new mail appears in your inbox the moment it arrives, with query-token auth for secure EventSource connections.

🛡️ Admin Panel & Teams

Super admin user management with per-user quotas on domains, inboxes, message size, and storage. Multi-user teams with fine-grained permissions — invite collaborators and control access across email inboxes, API keys, and websites.

How It Works

Get your email infrastructure running in minutes with three simple steps.

1

Deploy

Run a single install script on your Ubuntu/Debian VPS. Docker, PostgreSQL, Redis, nginx, TLS — all provisioned automatically.

2

Configure DNS

Point your domain's MX record to GoMail. Add SPF, DKIM, and DMARC records for maximum deliverability — our panel guides you step by step.

3

Start Receiving

Create inboxes, generate API keys, upload static sites — everything works immediately. Monitor real-time email flow from the dashboard.

Who Is GoMail For?

Built for developers, agencies, and businesses who want full control over their email infrastructure.

🖥️ Developers & SaaS Builders

Add email receiving to your application via REST API or SMTP. Programmatic inbox creation, webhook-ready real-time events, and scoped API keys for multi-tenant architectures.

🏢 Agencies & Freelancers

Host static websites for clients with custom domains and automatic SSL. Manage multiple projects from one dashboard. Deploy with a simple ZIP upload — no Git, no CI/CD needed.

🔒 Privacy-Focused Teams

Keep your email data on your own infrastructure. No third-party email providers reading your messages. End-to-end control with JWT auth, bcrypt password hashing, and session chain revocation.
